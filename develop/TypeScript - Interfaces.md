# Note
#### 什麼是 Interface ?
TypeScript 中的 `interface` 主要是用於描述複雜的資料結構，並且增強了程式碼的可重用性。它讓我們可以創建一種新的名稱，用於描述可能具有多個屬性和方法的物件結構與基本型別註記相比，`interface` 提供了一種方式，讓我們可以定義並重用更複雜的結構，對於開發大型或者結構複雜的專案非常有用。
