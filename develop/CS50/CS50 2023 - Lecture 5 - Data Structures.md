# Note
### Stacks and Queues
#### Queue
例如現實生活中的排隊，就是Queue
FIFO(First in first out)
- enqueue(adding, offering)：將資料放入佇列尾端
- dequeue(polling)：取出佇列前端之資料
#### Stacks
例如盤子堆疊，一個疊一個，照理會從最外層的盤子開始拿又或者是Gmail，預設新的信件都會在最上面
LIFO(Last in first out)
- push：將資料放在最頂端
- pop：從最頂端拿出資料
